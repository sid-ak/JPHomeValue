# JPHomeValue
Evaluates home value in the South Florida region.  
A project provided by [J.P. Morgan](https://www.jpmorgan.com/global) and [University of South Florida](https://www.usf.edu/).

## Stack
- Angular
- Firebase
- Python/TensorFlow
